ABOUT ME

I'm Electro, a 17 year old artist and programmer from Italy.

While I have been drawing for my whole life, I've only been programming for 3 years, starting with Gamemaker 8.1.
I've always been fascinated about making games and bringing my characters to life, always trying to improve my game development skills
as well as learning more professional programming languages, but I'm still very early into them.
Right now I'm learning more professional programming languages like Javascript, web development with things like this site, and even getting into making music.

You can contact me on Twitter or you can send me an email, I'll try to answer as soon as I can.
